Overview
---------
Markwhat is a desktop cross-platform markup text editor with live preview feature.
Written in pure ``python``, using ``Tkinter`` as its ``GUI`` framework.


Available parsers
-----------------

* ``Markdown``
* ``Textile``
* ``restructuredtext``
